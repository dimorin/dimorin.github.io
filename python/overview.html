# linux 기본/필수 명령어

명령어 명령어 사용설명 사용법(예시)
ls 현재 디렉토리의 파일 나열 ls -al 파일 상세정보까지 나열
cd 디렉토리 변경 cd .. 상위 디렉토리 이동, cd / 루트 디렉토리 이동
pwd 현재 디렉토리 알아보기 pwd
cp 파일 복사하기 cp 복사할파일 저장할 곳
mv 파일 이동하기 mv 이동할파일 저장할 곳
mkdir 디렉토리 생성 mkdir 디렉토리명
rm -r 디렉토리와 하위 폴더 파일 모두 삭제 rm -r 디렉토리명
passwd 사용자의 패스워드 변경 passwd
su 사용자 변경 sudo
reboot 시스템 다시 시작하기 reboot

# linux 편집기(editor)

편집기 종류 :

vi, vim, ...

기본적으로 라즈베리파이에 라즈비안 운영체제를 설치하면, vi 편집기를 사용할 수 있다.

나머지 편집기들은 apt-get install 명령어를 통해 설치해야 사용가능하다.

# python 쉘

라즈베리파이에서는 파이선 쉘 이라는 프로그램을 제공하여 이 툴을 통해 입력하는 명령어의 결과를 바로 확인할 수 있다.

# python 설치

윈도우용 파이썬 설치

https://www.python.org/downloads/windows/

설치 후,


python --version

파이썬 콘솔을 실행하기 위해, 윈도우에서는 python을 입력하고, 맥OS나 리눅스라면 python3를 입력하고 엔터를 입력하세요.


$ python

Python 3.6.1 (...)

Type "help", "copyright", "credits" or "license" for more information.

>>>

파이썬을 실행시키면 프롬프트가 >>>로 바뀝니다. 이제부터는 파이썬 언어 명령어만 사용할 수 있다는 뜻이예요. >>>를 입력할 필요 없어요. 파이썬이 대신해 줄 테니까요.

파이썬 콘솔에서 나가려면 exit()을 입력하거나, 윈도우에서는 Ctrl + Z를, 맥이나 리눅스에서는 Ctrl + D를 입력하면 됩니다. 그러면 >>> 는 사라질 거에요.

파이썬이 설치된 폴더( 예: C:\Users\username\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Python 3.7)로 이동 후,

test.py라는 파일을 만들고 내부에 다음과 같이 적는다.


print('hello')

test.py를 다음과 같이 실행시킨다.


python test.py
