# Vue CLI 3로 프로젝트 생성

Install:

npm install -g @vue/cli

Create a project:

vue create my-project

vue app 열기:

cd my-project
npm run serve

localhost:8080 에서 확인

Vue CLI 3

# 기존 vue cli 로 프로젝트 생성

위에서 만들어놓은 프로젝트 폴더 삭제

rm -rf my-project

webpack-simple 템플릿으로 my-project 프로젝트 생성

vue init webpack-simple my-projecct
cd my-project
npm i
npm run dev

localhost:8080 에서 확인
