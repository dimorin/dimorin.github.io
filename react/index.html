# React

리액트는 사용자 정의 태그를 만드는 기술이다.

Props는 컴포넌트의 외부에서 사용하는 입력값이다.

State는 컴포넌트 내부에서 사용하는 값이다.

이벤트는 Props를 이용한다.

event.preventDefault()를 이용해서 기본 동작을 비활성화 할 수 있다.

form을 변경할 때는 Props를 State로 전환 후 onChange를 이용한다. (Contorlled/Uncontrolled Component)
